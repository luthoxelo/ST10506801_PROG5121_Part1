# ST10506801_PROG5121_Part1

##Overview

This project is a simple Java console-based application that demonstrates a user registration and login system. It validates user input such as username, password, and South African phone number before allowing access.

##The program is divided into two main classes:

Login.java = Handles all validation and authentication logic
Main.java → Handles user interaction via the console
#Features
##User Registration

##The system checks the following:

###Username Requirements
-Must contain an underscore (_)
=Must not exceed 5 characters
=Password Requirements
=Minimum of 8 characters

###Must contain:
=At least one uppercase letter
=At least one number
=At least one special character
=Phone Number Requirements
=Must start with South African international code: +27
=Must be 12 characters long

#User Login
=User enters username and password
=System verifies credentials against stored values
=Displays appropriate success or error message
